Contributor Guide
=================

Prerequisites
-------------
To build the plugin, you need to install JDK 21.

For actual usage and automated testing, you also need to install AutoHotKey v2.

Build
-----
```console
$ ./gradlew buildPlugin
```

Test
----
```console
$ ./gradlew test
```

Run in Debug IDE
----------------
```console
$ ./gradlew runIde
```
